Propriedades Display e Position no CSS

Display - Conceito

A propriedade display define como um elemento aparece e se organiza na página.

block

Ocupa toda a largura

inline

Inline 1 Inline 2

O elemento fica na mesma linha que os outros e ocupa apenas o espaço do seu conteúdo.

Não quebra linha

Não aceita largura e altura

Exemplo: ,

inline-block

Box 1
Box 2

Mistura características de inline e block.

Fica na mesma linha

Aceita largura e altura

Muito usado para alinhar elementos lado a lado com controle de tamanho.

none

(Elemento escondido, não aparece)

Some completamente

Não ocupa espaço

Usado para esconder elementos

flex

Alinhamento horizontal ou vertical
Controle fácil de espaçamento

grid

Características:
Cria um sistema baseado em linhas e colunas.
Layout estruturado
Maior controle de posicionamento

Position - Conceito

A propriedade position define como um elemento é posicionado na página.

static

Valor padrão, segue o fluxo normal.

relative + absolute

Caixa pai
Absolute

O elemento se move em relação à sua posição original.

Mantém seu espaço original

Permite deslocamento com top, left, etc.

O elemento é posicionado em relação ao elemento pai mais próximo que tenha posição definida.

Sai do fluxo normal

Pode sobrepor outros elementos

Se não houver pai posicionado, usa o body como referência.

fixed

Fica fixo na tela

Não se move com o scroll

Muito usado em menus e botões fixos

sticky

Ele "gruda" no topo quando rolar
Botão Fixo

Conclusão

O display controla como os elementos aparecem. O position define onde eles ficam.
